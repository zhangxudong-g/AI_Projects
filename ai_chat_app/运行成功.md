## 📱 使用说明

### 1. 登录
- **用户名**: 任意（例如：admin）
- **密码**: 至少 6 位（例如：123456）
- 点击登录按钮

### 2. 主界面
底部导航栏：
- 💬 **对话** - AI 聊天
- 📄 **文档** - 文档生成
- 📜 **历史** - 历史记录
- ⚙️ **设置** - 应用设置

### 3. AI 对话
1. 在输入框输入消息
2. 点击发送或按回车
3. AI 会流式回复（逐字显示）
4. 支持 Markdown 格式
5. 点击复制按钮可复制消息

### 4. 文档生成
1. 选择文档类型（文章/报告/邮件/代码等）
2. 输入生成要求
3. 点击"生成文档"
4. 查看生成的内容

### 5. 设置
- 选择 AI 模型
- 配置 API 地址
- 开启/关闭流式输出

## 🛠️ 开发命令

### 运行应用
```bash
# 方法 1: 双击
start.bat

# 方法 2: 命令行
flutter run -d chrome
```

### 热重载
在运行终端按：
- `r` - 热重载
- `R` - 完全重启
- `q` - 退出

### 停止应用
关闭 Chrome 窗口或按 `q` 退出
