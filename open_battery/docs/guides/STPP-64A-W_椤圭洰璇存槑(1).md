## 一、项目概述

### 1.1 产品定位
- 产品名称：**ポポフLite (Portable Power Hub Lite)**
- 产品型号：**STPP-64A-W**
- 用途：为手机、耳机、智能手表等移动小型设备及移动PC充电（支持PD规格的Type-C充电）
- 可放置在学校或办公室的固定场所，便于整理整顿，兼具装饰品功能
- 内置一条充电线，能给笔记本电脑、平板电脑和手机充电
- 能通过单充电底座给主体充电（支持PD规格的Type-C充电）
- 支持给5个主体同时充电，充电底座匹配一台DC输出的适配器

### 1.2 系统架构

```
┌────────────────────────────────┐
       │         │               │
│          MCU: F06Q20 (4MHz)    │
│  ┌───────┐ ┌─────────┐         │
│  │ IIC1  │ │  IIC2    │        │
│  │SW6301 │ │ SW6303   │        │
│  └───┬───┘ └─┬──────┬ ┘        │
└──────┼───────┼      ┼──────────┘
       │       │      │
    C2 65W  C1 45W   A 18W
   双向PD    单向PD    QC3.0
```

**控制逻辑模块**:
- **MCU (F06Q20)**: 集中管理，通过二路独立软件IIC控制PMIC
- **SW6301**: 负责Type-C双向65W（C2口 IN/OUT）
- **SW6303**: 负责C1输出45W + A口输出18W

## 三、端口规格与输出能力

### 3.1 各端口独立输出

| 端口 | 位置 | 方向 | 最大功率 | 输出规格 | 支持协议 |
|------|------|------|----------|----------|----------|
| **C1** | 前方左侧 | 单向OUT | **45W** | 5V⎓3A / 9V⎓3A / 12V⎓3A / 15V⎓3A / 20V⎓2.25A | PD 3.0/2.0, QC3.0/2.0 Class B |
| **A** | 前方居中 | 单向OUT | **18W** | 5V⎓3A / 9V⎓2A / 12V⎓1.5A | QC3.0/2.0 Class B |
| **C2** | 前方右下 | **双向** IN/OUT | **65W** | 5V⎓3A / 9V⎓3A / 12V⎓3A / 15V⎓3A / 20V⎓3.25A | PD 3.0, QC3.0/2.0 Class B |

### 3.2 多口同时输出功率分配

| 组合 | C1输出 | C2输出 | A口输出 | 合计 |
|------|--------|--------|---------|------|
| **C1单独** | 45W (Max) | - | - | 45W |
| **C2单独** | - | 65W (Max) | - | 65W |
| **A单独** | - | - | 18W (Max) | 18W |
| **C1+C2** | 30W (Max) | 65W (Max) | - | 95W |
| **C1+A** | 共享18W (Max) | - | 共享18W (Max) | 18W |
| **C2+A** | - | 65W (Max) |  18W (Max)| 83W |
| **C1+C2+A** | 18W (Max) | 65W (Max) | - | 83W |

### 3.3 C2口IN功能软件可控

- 不需要IN功能时可通过软件关闭

## 四、充电底座规格

### 4.1 单体底座充电
| 项目 | 规格 |
|------|------|
| 输入口 | TYPE-C接口 TC1 |
| 输入规格 | 5V⎓3A / 9V⎓3A / 12V⎓2.5A / 15V⎓2A / 20V⎓1.5A（快充，最高30W） |
| 充电触点 | 主体下方中心小环嵌入大环，双环式充电触点（无电压时断开） |
| 检测机制 | 有主体在底座上时，底座才输出 |
| 启动电压 | 底座提供3.3V电压+10K电阻输出，主体需匹配下拉电阻，启动电压2.6V(2.4-2.8V) |
| 充电策略 | 当有C2充电时，采用先充先得原则 |

### 4.2 连体底座充电（充电托盘）
| 项目 | 规格 |
|------|------|
| DC输入 | DC5525接口，18V⎓10A |
| 输出口 | 5路，每路功率30W（Max） |
| 同时充电 | 可同时给5台主体充电 |
| 限制 | 托盘对应的本体只能用托盘充电，不能从本体C2充电（通过软件控制） |

## 五、按键功能

### 5.1 按键操作定义

| 操作 | 功能 |
|------|------|
| **单击** | 开机/唤醒：相应电量指示灯长亮，黄灯常亮，输出检测启动 |
| **长按10秒** | 关机：输出关断，单击按键即重新开机（工厂出货模式） |

### 5.2 各状态下的按键行为

| 状态 | 按键行为 |
|------|----------|
| 无负载不充电 | 单击→指示灯长亮+黄灯常亮→输出检测启动→10s后指示灯熄灭→24h后输出检测关闭→进入待机（关机） |
| 接入负载时 | 黄灯长亮，电量指示灯常亮显示剩余电量；负载断开后同第1点 |
| 主体充电时 | 相应电量的指示灯闪烁，黄灯常亮；充满后4个电量指示灯长亮+黄灯常亮→10s后熄灭全部指示灯 |
| 即将没电（低电） | 黄灯长亮，最后一个白灯慢闪 |
| 完全没电关机 | 白灯完全熄灭，红灯快闪，3秒后熄灭 |

## 六、LED指示灯规格

### 6.1 LED配置
- 共**5个LED指示灯**：1红黄（工作指示灯）+ 4白（电量指示）
- 左边第一个红黄灯是工作指示灯，开机黄常亮
- 其他4个白色LED显示实时电量
- 充电时相应电量指示灯闪动

### 6.2 电量指示映射表

| 电量范围 | 充电状态（闪烁表示正在充） | 非充电开机状态 |
|----------|---------------------------|----------------|
| **100-61%** | 4灯常亮(1黄+3白)，第5灯闪烁 | 5灯全亮(1黄+4白) |
| **60-31%** | 3灯常亮(1黄+2白)，第4灯闪烁 | 4灯亮(1黄+3白) |
| **30-11%** | 2灯常亮(1黄+1白)，第3灯闪烁 | 3灯亮(1黄+2白) |
| **10-3%** | 1灯常亮(1黄)，第2灯闪烁 | 1灯常亮(1黄)+第2灯亮 |
| **3-1%** | 红灯快闪3秒后熄灭，4白灯完全熄灭 | 同左 |

### 6.3 充放电指示灯行为

| 场景 | 指示灯行为 |
|------|------------|
| 主体不充电，C1/C2/A接入负载 | 自动放电，黄灯长亮，相应电量指示灯常亮；负载充满10s后指示灯熄灭，进入待机 |
| 主体通过底座充电 | C1/C2/A能自动放电，黄灯长亮，相应电量指示灯**闪烁**；主体和负载都充满，指示灯全亮10s后全灭 |
| 主体通过C2口充电 | C1/A能自动放电，黄灯长亮，相应电量指示灯**闪烁**；主体和负载都充满，指示灯全亮10s后全灭 |

## 七、温度保护（NTC）

### 7.1 温度保护阈值

| 保护类型 | 保护触发条件 | 恢复条件 |
|----------|-------------|----------|
| **充电高温保护** | ≥50℃ | ≤40℃ |
| **充电低温保护** | ≤0℃ | ≥0℃ |
| **放电高温保护** | ≥60℃ | ≤55℃ |
| **放电低温保护** | ≤-10℃ | ≥-5℃ |

### 7.2 温度保护行为
- 触发温度保护时，关闭充放电输出
- NTC告警期间LED闪烁提示
- 温度恢复到安全范围后自动恢复充放电

## 十一、软件架构与代码规范

### 11.1 目录结构
```
src/          — 所有活跃源代码（扁平布局）
keil/         — Keil µVision工程文件和构建产物
sdk/          — SINOMICON SDK文件（Inc_mcu.h, STARTUP_F06.A51）
legacy/       — 归档的旧硬件模块（SW6124时代）
docs/         — 项目文档和规格书
```

### 11.2 构建环境
- **编译器**: Keil C51 v9.59，目标MCS-51
- **闪存**: SINOMICON MPT51编程器
- **内存约束**: IROM 16KB, IRAM 256B, XRAM 768B
- **看门狗**: 启用（WDT_C = 0x20），主循环中频繁调用CLR_WDT

### 11.3 中断向量
| 向量 | 来源 | 处理函数 | 运行频率 |
|------|------|----------|----------|
| INT0 (0) | P06/P20/P13下降沿 | INT0Interrupt | 唤醒/插入检测 |
| INT1 (2) | TMR0溢出(1ms) | INT1Interrupt | 系统滴答：定时器、KEY_Scan()、LED_RUN() |
| INT5 (10) | UART0 RX | INT5Interrupt | 调试UART |

### 11.4 IIC通信
- 二路独立软件模拟IIC总线，无硬件IIC外设
- IIC1 (iic_1.c) → SW6301
- IIC2 (iic_2.c) → SW6303
- 全局变量`IIC_ADDRESS`/`IIC_DAT`跨所有调用者共享，不可重入
- 寄存器访问模式：设置地址→读取→修改→写回

## 十三、总结

该项目为一款多接口高功率移动电源方案，具备：
- 多协议快充能力（PD 3.0/2.0, QC3.0/2.0 Class B）
- Type-C双向快充（C2口65W）
- 多口输出控制（C1 45W + A 18W + C2 65W）
- MCU集中管理架构（F06020）
- 智能温度保护（NTC监控）
- 充电底座支持（单体+连体托盘）
- 完善的LED电量指示系统
- 适用于消费电子供电、便携电源等场景
