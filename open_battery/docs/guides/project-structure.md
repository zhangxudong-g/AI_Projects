# 项目结构与代码架构

## 项目概述

mini-Lite 移动电源固件，运行在 SH68F06 (8051 内核, 16MHz) 上，通过 IIC 总线控制 SW6301（升压输出）和 SW6303（充电管理）。

## 目录结构

```
AI 电源/
├── arduino_iic_simulator/      # Arduino IIC 模拟器（开发调试用）
├── demo/                        # 固件项目
│   ├── TMR1_111/                # 主固件
│   │   ├── src/                 # 应用层代码
│   │   │   ├── main.c/h         # 主程序、IO/ADC初始化、主循环
│   │   │   └── int.c            # 中断处理（定时器、按键、IRQ）
│   │   ├── prj/                 # Keil工程 + 自写驱动
│   │   │   ├── test_tmr1.uvproj # Keil 项目文件
│   │   │   ├── IIC.c/h          # IIC 通信驱动
│   │   │   ├── LED.c/h          # LED 显示控制
│   │   │   ├── KEY.c/h          # 按键处理
│   │   │   ├── sys.c/h          # 系统函数
│   │   │   └── sleep.c          # 功耗管理
│   │   └── user_com/            # 厂商通信模块
│   │       └── user_com.c/h
├── docs/                        # 项目文档
│   ├── datasheets/              # 芯片数据手册
│   └── guides/                  # 开发指南
└── CLAUDE.md                    # AI 辅助开发配置
```

## 模块职责

```
┌─ 应用层 (src/) ─────────────────────────────────────────┐
│  main.c    系统初始化、主循环、按键逻辑、LED扫描          │
│  int.c     1ms定时中断、按键扫描、时间计数                │
└──────────────────────────┬───────────────────────────────┘
                           │
┌─ 驱动层 (prj/) ──────────┴───────────────────────────────┐
│  sys.c     充放电状态机 (Set_Power)、电压/电流判断        │
│  LED.c     电量显示、三线矩阵扫描                         │
│  KEY.c     按键消抖、长按/短按识别                        │
│  sleep.c   低功耗睡眠模式                                 │
│  IIC.c     软件模拟 IIC 底层协议                          │
└──────────────────────────┬───────────────────────────────┘
                           │ IIC 总线
┌─ 芯片层 (外部) ──────────┴───────────────────────────────┐
│  SW6301    升压输出IC (地址 0x14B, 5-12V 输出)           │
│  SW6303    充电管理IC (地址 0x23, 12V 输入)              │
└──────────────────────────────────────────────────────────┘
```

## GPIO 引脚分配

| 引脚 | 方向 | 功能 | 说明 |
|------|------|------|------|
| P10 | I/O | SDA | IIC 数据线 |
| P07 | OUT | SCL | IIC 时钟线 |
| P06 | IN | IRQ1 | SW6303 中断，下降沿 |
| P20 | IN | IRQ2 | SW6301 中断，下降沿 |
| P21 | IN | 充电检测 | 低电平=有负载 |
| P13 | IN | 按键 | 下降沿中断 + 定时扫描 |
| P17 | OUT | LED1 | 电量显示 |
| P12 | OUT | LED2 | 电量显示 |
| P11 | OUT | LED3 | 电量显示 |
| P14 | OUT | 照明控制 | 充电/照明 MOS |
| P02 | OUT | UART TX | 调试串口输出 |

## 文件功能索引

| 文件 | 核心函数 | 功能 | 修改频率 |
|------|---------|------|---------|
| main.c | `main()`, `IO_INIT()`, `KEY_RUN()`, `LED_RUN()` | 系统初始化、主循环、按键、LED | 高 |
| int.c | `INT1Interrupt()` | 1ms 定时中断、时间计数 | 低 |
| sys.c | `Set_Power()`, `PowerOn_Chk()` | 充放电状态机、电压电流判断 | 高 |
| IIC.c | `IIC_WriteByte()`, `IIC_ReadByte()` | IIC 底层通信 | 低 |
| LED.c | `LED_RUN()` | 三线矩阵 LED 扫描 | 中 |
| KEY.c | `KEY_Scan()` | 按键消抖、长/短按 | 中 |
| sleep.c | `SLEEP_MD` | 睡眠模式进入 | 低 |
| user_com.c | 厂商提供 | 通信功能封装 | 低 |
