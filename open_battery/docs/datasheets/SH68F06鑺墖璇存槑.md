# SH68F06 芯片说明

## 概述

SH68F06 是一颗基于 **8051 内核的 8位微控制器 (MCU)**，专用于消费电子产品，如移动电源（充电宝）等。在本项目中，SH68F06 作为 **mini-Lite 移动电源的主控芯片**，协调充电和放电过程，提供用户界面，并实现各种保护功能。

## 主要功能特性

### 1. 核心功能 - mini-Lite 的"大脑"

```
电池 ──► [SW6301] ──► 输出        (升压输出IC)
         ↑ 通过 IIC 控制

电池 ◄── [SW6303] ◄── 输入        (充电管理IC)
         ↑ 通过 IIC 控制

         SH68F06 (主控)
         通过 IIC 控制两个芯片
```

SH68F06 的核心作用：
- **控制 SW6301** (升压芯片) - 设置输出电压 5V/9V/12V
- **控制 SW6303** (充电芯片) - 设置充电电流，读取充电状态
- **管理 LED 显示** - 显示电量、充电状态
- **检测按键输入** - 用户交互
- **保护逻辑** - 过温、过流、低电量保护

### 2. 硬件接口定义

| 功能 | Pin | 方向 | 说明 |
|------|-----|------|------|
| LED1 | P16 | 输出 | 电量指示灯 |
| LED2 | P15 | 输出 | 电量指示灯 |
| LED3 | P14 | 输出 | 电量指示灯 |
| LED4 | P13 | 输出 | 电量指示灯 |
| DC_EN | P11 | 输出 | 升压使能控制 |
| SDA | P10 | 双向 | IIC 数据线 |
| SCL | P07 | 输出 | IIC 时钟线 |
| 按键 | P00 | 输入 | 电源按键 |
| 无线充电检测 | P06 | 输入 | 外部中断 |
| 底座检测 | P21 | 输入 | 外部中断 |

#### 代码定义示例

```c
// LED 定义 (来自 LED.h)
#define LED1  	P16   // 电量指示1
#define LED2  	P15   // 电量指示2
#define LED3  	P14   // 电量指示3
#define LED4  	P13   // 电量指示4
#define DC_EN   P11   // 升压使能

// IIC 定义 (来自 IIC.h)
#define SCL  P07       // IIC 时钟线
#define SDA  P10       // IIC 数据线
#define Wireless_charging  P06
#define IRQ  P06

// 按键定义 (来自 KEY.h)
#define P_Key P00      // 电源按键
```

### 3. 内置外设功能

#### ADC (模数转换器)

功能：
- **电压采样** - 检测电池电压
- **温度检测** - NTC 热敏电阻采样
- **多通道 ADC** - 支持多个模拟输入

```c
// ADC 初始化和使用示例
void ADC_INIT(void)
{
    VREF_EN;             // ADC参考使用配置
    ADC_EN;
    ADC_RVDD_AVG8;      // ADC 内部2.5V参考
    ADC_VDD_DIV4;       // 选择通道0
}

// 读取 ADC 值
unsigned int GET_ADC_VALUE(void)
{
    VREF_EN;
    ADC_EN;
    ADC_RVREF25_AVG8;     // ADC 内部2.5V参考
    ADC_P16_AN10;         // 选择通道
    DelayUs(1000);
    ADC_SOFT_TG;          // ADC启动
    ADC_WAIT;             // 等待转换完成
    tmp_data = (ADC_DH << 8) | ADC_DL;
    ADC_INT_IF_CLR;       // 清中断标志位
    return tmp_data;
}
```

#### UART (串口通信)

```c
void UART_INIT(void)
{
    P02_SET;
    P02_FUN_TX0;         // P02 设置为 UART 发送
    P02_OUT_EN;          // 该管脚设置成输出
    
    UART0_RX8;           // 八位数据接收
    UART0_TX8_1S;        // 发送一个停止位
    UART0_BRR_WR(34);    // 波特率设置 (9600)
    UART0_RX0_INT_EN;    // 接收中断使能
    INT5_EN;             // UART 所属中断使能
    GIE_EN;              // 总中断开启
}
```

#### 中断系统

中断源：
- **按键中断** (P00)
- **IIC 中断** (P06)
- **底座检测中断** (P21)

```c
// 中断初始化示例
void IO_INIT(void)
{
    P06_IN_EN;   // IRQ
    P06_PU_EN;   // 上拉
    P06_INT_IF_CLR;
    P06_NEG_INT_EN;       // 下降沿触发
    
    P20_IN_EN;   // IRQ
    P20_PU_EN;   // 上拉
    P20_INT_IF_CLR;
    P20_NEG_INT_EN;       // 下降沿触发
    
    P13_IN_EN;   // 按键
    P13_PU_EN;   // 上拉
    P13_INT_IF_CLR;
    P13_NEG_INT_EN;       // 下降沿触发
    
    INT0_EN;
}
```

#### 软件模拟 IIC

由于 SH68F06 的 IIC 使用软件模拟，通过 GPIO 实现：

```c
// IIC 通信函数

void IIC_Stop(void);    // 发送停止信号
void SendByte(u8 dat);  // 发送一字节

// 寄存器读写

```

## C51 编程规范

### 位操作

SH68F06 是 8051 内核，支持强大的位操作功能：

```c
// 使用 sbit 定义位变量
sbit LED1 = P1^6;
sbit DC_EN = P1^1;

// 位寻址区：20H-2FH（128位），支持直接位操作
P1 |= 0x01;    // 置位
P1 &= ~0x01;   // 复位
```

### 中断处理

中断向量表位于：
- `03H` (INT0)
- `0BH` (T0)
- `1BH` (T1) 等

中断服务函数声明：
```c
void INT0_Isr(void) interrupt 0
```

中断优先级通过 IP 寄存器配置。注意中断现场保护和恢复（ACC、PSW 等）。

### 工作模式

- **充电模式**：插入输入电源时进入，LED 显示充电状态
- **放电模式**：按键触发或负载插入时进入，升压输出
- **睡眠模式**：低电量或无操作时进入，降低功耗
- **保护模式**：过温、过流、欠压时触发

## 调试技巧

### 使用 Arduino 模拟器调试

1. 上传 Arduino 模拟器
2. 打开串口监视器 (115200)
3. 观察 IIC 通信日志

示例输出：
```
[SW6301] 写入: 0x02 = 0x4B
    → 设置输出电压: 5V
```

### 常见问题

| 问题 | 可能原因 | 解决方法 |
|------|---------|---------|
| 无 ACK 应答 | 芯片地址错误 | 检查 IIC 地址定义 |
| 读取数据错误 | 时序不匹配 | 调整延时参数 |
| SDA/SCL 无信号 | GPIO 配置错误 | 检查 SDA_OUT/SDA_IN |

## 总结

**SH68F06 是一颗低成本的 8051 内核微控制器**，专门用于移动电源等消费电子产品。它具备：

✅ **足够的性能** - 控制两个电源芯片，处理用户输入  
✅ **丰富的外设** - ADC、UART、GPIO、中断  
✅ **低功耗** - 适合电池供电设备  
✅ **成本低廉** - 适合大规模生产  

在你的项目中，SH68F06 是整个移动电源系统的**控制核心**，协调充电和放电过程，提供用户界面，并实现各种保护功能。
