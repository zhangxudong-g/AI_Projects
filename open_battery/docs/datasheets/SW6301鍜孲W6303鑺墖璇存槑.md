# SW6301 和 SW6303 芯片说明

## 概述

SW6301 和 SW6303 是**电源管理芯片**，用于移动电源（充电宝）中。

---

## SW6301 - 升压输出芯片 (Boost IC)

**功能**：将电池电压升压到 5V/9V/12V 输出给设备充电

```
电池 (3.7V) ──► SW6301 ──► 输出 (5V/9V/12V)
                       升压
```

**主要特性**：
- 支持多种输出电压（5V/9V/12V）
- 通过 IIC (0x14B) 控制输出电压和电流
- 具有过流、过温、短路保护

**IIC 地址**：
- 10 位地址：0x14B
- 7 位格式：0x0A

**寄存器定义**：

| 地址 | 名称 | 说明 | 取值 |
|------|------|------|------|
| 0x02 | OUTPUT_VOL | 输出电压 | 0x4B=5V, 0x6B=9V, 0x7B=12V |
| 0x03 | OUTPUT_CUR | 输出电流 | 值 × 100mA |
| 0x04 | STATUS | 状态寄存器 | 各位表示不同状态 |

**应用场景**：
- 手机快充（9V/12V）
- 平板电脑充电
- 标准 USB 设备供电（5V）

---

## SW6303 - 充电管理芯片 (Charger IC)

**功能**：管理输入电源对电池的充电

```
输入 (5V-12V) ──► SW6303 ──► 电池充电
                     充电管理
```

**主要特性**：
- 支持多种输入电压
- 通过 IIC (0x23) 控制充电电流和状态
- 充电状态监测（充电中、充满、异常）

**IIC 地址**：
- 7 位地址：0x23

**寄存器定义**：

| 地址 | 名称 | 说明 | 取值 |
|------|------|------|------|
| 0x20 | INPUT_VOL | 输入电压设置 | bit0=1 则为 12.6V |
| 0x21 | CHG_CUR | 充电电流 | 值 × 100mA |
| 0x22 | STATUS | 充电状态 | 0x01=充电中 |

**应用场景**：
- 适配器输入充电
- USB 输入充电
- 充电状态指示

## 在 mini-Lite 中的角色

```
┌──────────────────────────────────────┐
│         mini-Lite 移动电源           │
│                                      │
│   输入 ──► [SW6303] ──► 电池        │
│            充电IC                   │
│                                      │
│   电池 ──► [SW6301] ──► 输出        │
│            升压IC                   │
│                                      │
│         SH68F06 (主控)               │
│         通过 IIC 控制两个芯片         │
└──────────────────────────────────────┘
```

**简单说**：
- **SW6303** = 给移动电源充电（输入）
- **SW6301** = 给手机/设备充电（输出）

## IIC 通信

### 地址格式

| 芯片 | 10位地址 | 7位格式 | 说明 |
|------|---------|---------|------|
| SW6301 | 0x14B | 0x0A | 需右移1位 |
| SW6303 | 0x23 | 0x23 | 标准7位地址 |

### 通信函数

```c
// IIC 驱动函数（在 IIC.c 中实现）
void IIC_Start(void);       // 起始信号
void IIC_Stop(void);        // 停止信号
void SendByte(u8 dat);      // 发送字节
u8 RcvByte(void);           // 接收字节

// 寄存器读写
void IIC_WriteByte(void);   // 写寄存器
void IIC_ReadByte(void);    // 读寄存器
```

### 示例代码

```c
// 设置 SW6301 输出 9V
R_Address = 0x02;    // OUTPUT_VOL 寄存器
R_Value = 0x6B;      // 9V
IIC_WriteByte();

// 读取 SW6303 充电状态
R_Address = 0x22;    // STATUS 寄存器
IIC_ReadByte();
// R_Value & 0x01 判断是否充电中
```
