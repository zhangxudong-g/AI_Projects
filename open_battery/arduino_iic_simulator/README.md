# mini-Lite IIC 模拟器使用指南

## 功能说明

这个 Arduino 程序模拟了 mini-Lite 项目中的两个关键芯片：
- **SW6301**: 升压输出 IC (IIC 地址: 0x14B)
- **SW6303**: 充电管理 IC (IIC 地址: 0x23)

## 硬件连接

```
┌─────────────────┐              ┌─────────────────┐
│   SH68F06       │              │   Arduino       │
│   (主控)        │              │   (模拟器)      │
│                 │              │                 │
│  SDA ───────────┼─────────────►│  A4 (SDA)       │
│  SCL ───────────┼─────────────►│  A5 (SCL)       │
│  GND ───────────┼─────────────►│  GND            │
└─────────────────┘              └─────────────────┘
                                  USB
                                   │
                                   ▼
                              电脑串口监控
```

## 上传步骤

1. 打开 Arduino IDE
2. 加载 `arduino_iic_simulator.ino`
3. 选择开发板：Arduino Uno/Nano
4. 点击上传
5. 打开串口监视器 (115200 baud)

## 串口命令

### 基本命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `help` 或 `h` | 显示帮助 | `help` |
| `status` 或 `s` | 显示寄存器状态 | `status` |
| `stats` | 显示通信统计 | `stats` |
| `clear` | 清除统计信息 | `clear` |

### 高级命令

**设置寄存器值**
```
set <芯片> <寄存器地址> <值>

示例:
set sw6301 0x02 0x4B    # 设置 SW6301 输出 5V
set sw6303 0x20 0x20    # 设置 SW6303 输入 12V
```

**模拟事件**
```
sim <事件>

可用事件:
  normal     - 正常状态
  overtemp   - 过温保护
  overcurr   - 过流保护
  no_battery - 无电池
  charging   - 充电中
```

## 寄存器说明

### SW6301 寄存器

| 地址 | 名称 | 说明 |
|------|------|------|
| 0x02 | OUTPUT_VOL | 输出电压 (0x4B=5V, 0x6B=9V, 0x7B=12V) |
| 0x03 | OUTPUT_CUR | 输出电流 (值×100mA) |
| 0x04 | STATUS | 状态寄存器 |

### SW6303 寄存器

| 地址 | 名称 | 说明 |
|------|------|------|
| 0x20 | INPUT_VOL | 输入电压 (bit0=1则为12.6V) |
| 0x21 | CHG_CUR | 充电电流 |
| 0x22 | STATUS | 充电状态 (0x01=充电中) |

## 使用示例

### 场景1：监控通信

1. 上传程序到 Arduino
2. 打开串口监视器
3. 运行 SH68F06 固件
4. 观察串口输出：

```
[SW6301] 读取: 0x02 = 0x4B
[SW6301] 写入: 0x02 = 0x6B
    → 设置输出电压: 9V
```

### 场景2：模拟过温保护

```
> sim overtemp
→ 模拟: 过温保护
```

然后 SH68F06 会读到异常状态，触发保护逻辑。

### 场景3：查看统计

```
> stats

=== 通信统计 ===
SW6301:
  读取: 152
  写入: 23
SW6303:
  读取: 89
  写入: 12
```

## 注意事项

1. **地址冲突**: Arduino 标准库同一时间只能作为一个 IIC 从设备
2. **解决方法**: 需要使用第三方库或动态切换地址
3. **上拉电阻**: 确保 IIC 总线有 4.7kΩ 上拉电阻

## 故障排除

**问题**: SH68F06 无法通信
- 检查接线 (SDA/SCL 是否接反)
- 检查地址是否正确
- 检查上拉电阻

**问题**: 串口无输出
- 确认波特率设置为 115200
- 检查 USB 连接
- 尝试重启 Arduino
