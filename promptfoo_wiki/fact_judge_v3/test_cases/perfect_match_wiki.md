# Perfect Match Test

This class provides basic arithmetic operations for testing purposes.

## Methods

- `add(int a, int b)`: Adds two numbers together and returns the sum
- `subtract(int a, int b)`: Subtracts the second number from the first and returns the difference  
- `multiply(int a, int b)`: Multiplies two numbers together and returns the product

## Purpose

This class serves as a simple calculator for testing the evaluation system.
