# 边界测试用例集合

## 测试目标
- 验证系统在各种边界条件下的表现
- 检查极端情况下的评估准确性
- 确保系统稳定性和鲁棒性

## 测试类别

### 1. 空内容测试
- 空源代码 + 空Wiki
- 空源代码 + 任意Wiki
- 任意源代码 + 空Wiki

### 2. 极简内容测试
- 单行代码 + 单行Wiki
- 最小可行代码 + 最小可行Wiki

### 3. 极端内容测试
- 大量代码 + 空Wiki
- 空代码 + 大量Wiki
- 大量代码 + 大量无关Wiki

### 4. 特殊字符测试
- 包含特殊字符的代码和Wiki
- Unicode字符测试
- 控制字符测试

### 5. 语言边界测试
- 不支持的语言
- 混合语言代码
- 语法错误的代码

### 6. 伪造内容测试
- 编造不存在的函数/方法
- 编造不存在的业务逻辑
- 夸大代码功能

### 7. 边界分数测试
- 理论最高分测试
- 理论最低分测试
- 各种中间分数测试

### 8. 性能边界测试
- 超大文件测试
- 长时间运行测试
